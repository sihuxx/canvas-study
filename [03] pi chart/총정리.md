# 🥧 Canvas 파이 그래프 핵심 정리

---

## 1. 파이 그래프 필수 재료

```js
const total = data.reduce((acc, cur) => acc + cur.v, 0)  // 전체 합
const cx = canvas.width / 2   // 원 중심 x
const cy = canvas.height / 2  // 원 중심 y
const radius = 200             // 반지름
let current = -Math.PI / 2    // 시작 각도 (12시 방향)
```

---

## 2. 조각 그리기 핵심 공식

```js
// 값 → 각도 변환 (핵심!)
const angle = (v / total) * Math.PI * 2
// 전체(Math.PI * 2 = 360도) 중 내 비율만큼

ctx.beginPath()
ctx.moveTo(cx, cy)                              // 중심으로 이동
ctx.arc(cx, cy, radius, current, current + angle)  // 호 그리기
ctx.fill()

current += angle  // 누적 (없으면 모든 조각이 같은 자리에 겹침!)
```

---

## 3. 시작 각도

```
-Math.PI / 2  →  12시 ✅ (자연스러움)
-Math.PI      →  9시
0             →  3시 (arc 기본값)
```

---

## 4. 애니메이션

`angle * progress` 를 쓰면 조각이 동시에 커지는 효과

```js
ctx.arc(cx, cy, radius, current, current + angle * progress)
current += angle * progress  // current도 같이 progress 곱해야 함!
```

> `current`도 `* progress` 안 하면 조각들이 벌어지오!

---

## 5. 라벨 위치 계산 (cos, sin)

```js
const midAngle = current + (angle * progress) / 2  // 조각 중간 각도

const lx = cx + Math.cos(midAngle) * (radius * 0.65)  // x = 중심 + cos * 거리
const ly = cy + Math.sin(midAngle) * (radius * 0.65)  // y = 중심 + sin * 거리
```

```
cos/sin: 각도 → 좌표  (방향 × 거리 = 좌표)

radius * 0.65  →  원 안쪽
radius * 1.0   →  원 테두리
radius * 1.2   →  원 바깥
```

---

## 6. 호버 이벤트

### 원 안에 있는지 (피타고라스)
```js
const rect = canvas.getBoundingClientRect()
// 캔버스가 브라우저 어디에 있는지 가져옴
// 캔버스가 (0,0)에 없을 수 있어서 보정용으로 필요

const dx = e.clientX - rect.left - cx
// e.clientX   = 브라우저 왼쪽 끝에서 마우스까지 거리
// - rect.left = 캔버스 시작점 빼서 캔버스 기준으로 변환
// - cx        = 원 중심 빼서 중심 기준으로 변환
// 결과: 원 중심에서 마우스까지 x 거리

const dy = e.clientY - rect.top - cy
// dx랑 동일한 방식, y축 버전

Math.sqrt(dx**2 + dy**2) > radius
// 피타고라스: √(dx² + dy²) = 원 중심에서 마우스까지 실제 거리
// 이 거리가 radius보다 크면 원 밖 → 호버 해제
```

### 몇 번째 조각인지 (atan2)
```js
let mouseAngle = Math.atan2(dy, dx)
// atan2 = cos/sin 반대! 좌표 → 각도
// cos/sin: 각도 넣으면 좌표 나옴
// atan2:   좌표 넣으면 각도 나옴
// 반환값 범위: -π ~ π

if (mouseAngle < -Math.PI / 2) mouseAngle += Math.PI * 2
// 시작각도가 -π/2 (12시) 인데
// atan2가 -π/2 보다 작은 음수를 반환하는 구간이 있어서
// π*2 를 더해서 양수로 보정해줌

let current = -Math.PI / 2  // render와 동일한 시작각도!
data.forEach(({ v }, i) => {
  const angle = (v / total) * Math.PI * 2

  if (mouseAngle >= current && mouseAngle < current + angle) hoverIndex = i
  // current           = 이 조각 시작 각도
  // current + angle   = 이 조각 끝 각도
  // 마우스 각도가 이 범위 안에 있으면 이 조각이 호버된 것!

  current += angle  // 다음 조각 시작각도 = 이 조각 끝각도
})
```

| 함수 | 방향 | 용도 |
|------|------|------|
| `cos/sin` | 각도 → 좌표 | 라벨 위치 계산 |
| `atan2` | 좌표 → 각도 | 마우스가 몇 번째 조각인지 |
| `sqrt` | 거리 계산 | 마우스가 원 안인지 |

---

## 7. 호버 강조 (globalAlpha)

```js
// 호버 안된 조각 반투명
ctx.globalAlpha = (hoverIndex === -1 || i === hoverIndex) ? 1 : 0.5
ctx.fill()
ctx.globalAlpha = 1  // 반드시 초기화! 안하면 텍스트도 흐려짐
```

---

## 8. 기초 → 심화 단계

```
① 기초: 조각 그리기
   total → angle 계산 → arc() → current 누적

② 중급: 애니메이션
   angle * progress → current도 * progress

③ 중급: 라벨
   midAngle → cos/sin으로 원 안쪽 좌표

④ 심화: 호버
   sqrt로 원 안/밖 → atan2로 각도 → 조각 범위 비교

⑤ 심화: 호버 강조
   globalAlpha로 반투명 처리
```
