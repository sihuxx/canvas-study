# 📊 Canvas 바 그래프 핵심 정리

---

## 1. 기본 메서드

### 막대 그리기
```js
ctx.fillStyle = "#4A90D9"          // 색상 지정
ctx.fillRect(x, y, width, height)  // 막대 그리기
// height가 음수면 위쪽으로 그려짐 (바닥에서 위로 올라가는 막대)
```

### 선 그리기 (축)
```js
ctx.strokeStyle = "#ccc"  // 선 색상
ctx.beginPath()           // 새 경로 시작 (필수!)
ctx.moveTo(x1, y1)        // 시작점
ctx.lineTo(x2, y2)        // 끝점
ctx.stroke()              // 실제로 선 그리기
```

### 텍스트 그리기
```js
ctx.font = "20px sans-serif"
ctx.textAlign = "center"           // left | center | right
ctx.fillStyle = "#888"
ctx.fillText("텍스트", x, y)
```

### 매 프레임 초기화 (필수!)
```js
ctx.clearRect(0, 0, canvas.width, canvas.height)
// 이걸 안 하면 이전 프레임 위에 겹쳐서 그려짐
```

## 2. 바 그래프 레이아웃 공식

```
canvas.width
|←──────────────────────────────────────────→|
|← paddingLeft →|← slotW →|← slotW →|← slotW →|
                 |←gap→|bar|←gap→|bar|←gap→|bar|
```

### 핵심 계산
```js
const sectionWidth = (canvas.width - padding) / data.length
// 각 막대가 차지하는 구역 너비

// x + width = sectionWidth * (i+1) 이 되게만 만들면 정렬이 맞음!
const x     = sectionWidth * i + gap      // 막대 x 시작위치
const width = sectionWidth - gap          // 막대 너비
// x + width = sectionWidth * i + gap + sectionWidth - gap
//           = sectionWidth * (i+1) ✅
```

### 막대 중앙 정렬 (slotW 방식)
```js
const slotW = (width - paddingLeft) / data.length  // 슬롯 전체 너비
const barW  = slotW * 0.55                          // 막대는 슬롯의 55%

const x = paddingLeft + slotW * i + (slotW - barW) / 2
//                                   ↑ 슬롯 안에서 가운데 정렬
```

### 높이 계산
```js
// 값을 캔버스 높이 비율로 변환
const bh = (value / max) * (height - paddingBottom - paddingTop)

// y 좌표 (위에서 얼마나 내려올지)
const y = paddingTop + (height - paddingBottom - paddingTop) - bh
// => 바닥 기준선에서 bh만큼 위로 올라간 위치
```

### max 기준 비율 변환
```js
function valueToHeight(value) {
  return (canvas.height - padding * 2) / max * value
  // 최대값 대비 비율로 높이 계산
  // max(700) → 전체 높이 / 다른 값들 → 비례해서 작아짐
}
```

## 3. 애니메이션 패턴 (외워두면 평생 씀!)

```js
const duration = 1000  // 애니메이션 시간 (ms)
let startTime

function animate(timestamp) {
  // timestamp: 브라우저가 자동으로 넣어주는 값
  // 페이지 로드 후 경과 밀리초 (performance.now()와 동일)

  if (!startTime) startTime = timestamp       // 최초 1회 시작시간 저장
  const elapsed  = timestamp - startTime      // 경과 시간
  const progress = Math.min(elapsed / duration, 1)  // 0.0 ~ 1.0

  render(progress)

  if (progress < 1) requestAnimationFrame(animate)  // 1 되면 종료
}

requestAnimationFrame(animate)  // 최초 실행
```

### 타임라인 흐름
```
호출               timestamp    elapsed    progress
────────────────────────────────────────────────────
animate(1000)  →   1000ms       0ms        0.00  ← startTime 저장
animate(1016)  →   1016ms       16ms       0.016
animate(1032)  →   1032ms       32ms       0.032
...
animate(2000)  →   2000ms       1000ms     1.00  ← 종료
```

### progress 활용 예시
```js
// 막대 높이
height * progress

// 투명도 (페이드인)
ctx.globalAlpha = progress

// 위치 이동
x = startX + (endX - startX) * progress

// 크기 변화
radius = maxRadius * progress
```

> **핵심:** `render(progress)` 안에서 `progress(0→1)` 를 어디에 곱하냐만 바꾸면 어떤 애니메이션이든 구현 가능!
