# 🌹 Canvas 로즈 차트 핵심 정리

---

## 1. 파이 vs 로즈 차이

| | 파이 그래프 | 로즈 그래프 |
|--|------------|------------|
| 각도 | 값에 따라 다름 | 전부 균등 |
| 반지름 | 전부 동일 | 값에 따라 다름 |
| 각도 계산 | `(v / total) * Math.PI * 2` | `Math.PI * 2 / data.length` |
| 반지름 계산 | `radius` 고정 | `(v / max) * radius` |

---

## 2. 필수 재료

```js
const max = Math.max(...data.map(d => d.v))  // 최댓값 (반지름 비율 계산용)
const angle = Math.PI * 2 / data.length      // 균등하게 나눈 각도 (고정!)
const cx = canvas.width / 2
const cy = canvas.height / 2
const radius = 200
```

> 파이는 `total`이 필요, 로즈는 `max`가 필요!

---

## 3. 핵심 공식

```js
// 각도: 전부 동일하게 균등 분할
const angle = Math.PI * 2 / data.length

// 반지름: 값이 클수록 길어짐
const r = (v / max) * radius
```

---

## 4. 그리기 패턴

```js
let current = -Math.PI / 2  // 12시 방향에서 시작

data.forEach(({ v, c }) => {
  const r = (v / max) * radius  // 반지름이 값에 따라 다름

  ctx.beginPath()
  ctx.moveTo(cx, cy)
  ctx.arc(cx, cy, r, current, current + angle)  // 각도는 고정
  ctx.fillStyle = c
  ctx.fill()

  current += angle  // 균등하게 누적
})
```

---

## 5. 애니메이션

돌아가면서 + 반지름도 커지는 효과

```js
const r = (v / max) * radius * progress        // 반지름도 커지고
ctx.arc(cx, cy, r, current, current + angle * progress)  // 각도도 커짐
current += angle * progress                    // current도 같이 progress 곱하기!
```

> `current`에 `* progress` 안 하면 조각들이 벌어지오!

---

## 6. 라벨 위치

```js
const midAngle = current + (angle * progress) / 2  // 조각 중간 각도
const lx = cx + Math.cos(midAngle) * (r * 0.65)    // r 기준으로 안쪽
const ly = cy + Math.sin(midAngle) * (r * 0.65)
```

> 파이는 `radius * 0.65` 고정, 로즈는 **`r * 0.65`** (조각마다 반지름이 다르기 때문)

---

## 7. 자주 하는 실수

| 실수 | 결과 | 해결 |
|------|------|------|
| `r`에 `angle` 곱함 | 반지름이 이상하게 나옴 | `radius` 곱하기 |
| `d.value`로 접근 | `max`가 NaN | 데이터 키 확인 (`d.v`) |
| `current` 누적 안 함 | 모든 조각이 겹침 | `current += angle * progress` |
| `radius` 고정으로 arc 그림 | 로즈가 아닌 파이처럼 나옴 | `r` 사용하기 |
| 라벨을 `radius` 기준으로 계산 | 라벨이 조각 밖으로 나감 | `r * 0.65` 사용 |
